# Клиентское QML-приложение для Обучающей Платформы

Это репозиторий клиентского приложения для интерактивной обучающей платформы. Приложение разработано с использованием QML (Qt Modeling Language) и предоставляет интуитивно понятный пользовательский интерфейс для взаимодействия с серверной частью, где размещаются курсы, видеоуроки и профили пользователей.

## Описание Проекта

Клиентское QML-приложение является ключевым компонентом обучающей платформы, позволяющим пользователям:
*   Регистрироваться и авторизовываться в системе.
*   Просматривать каталог курсов, применять фильтры и поиск.
*   Получать доступ к подробной информации о курсах и видеоуроках.
*   Просматривать видеоуроки с отслеживанием прогресса.
*   Управлять своим профилем, включая личную информацию, контакты, аватар и обложку.
*   (Для авторов) Создавать и редактировать курсы, главы и загружать видео.
*   (Для административного персонала) Доступ к специфическим панелям управления (если предусмотрено клиентским интерфейсом).

## Структура Проекта (QML-файлы)

Основные компоненты пользовательского интерфейса расположены в директории `QML/`:

*   **`AuthPage.qml`**: Формы для регистрации и входа в систему, включая возможную интеграцию с OAuth (например, Google).
*   **`MainPage.qml`**: Главный экран приложения с навигацией и общей информацией.
*   **`MainSearchPage.qml`**: Интерфейс для поиска и фильтрации курсов по различным критериям.
*   **`CardVideoListView.qml`**: Компонент для отображения списков видеоуроков (например, в главах курса).
*   **`ProfilePage.qml`**: Детальное представление профиля пользователя с возможностью редактирования личных данных, контактов, аватара и обложки.
*   **`ProfileСhannel.qml`**: Отображение информации о канале или профиле автора, включая список его курсов.
*   **`SettingsPage.qml`**: Страница для управления настройками приложения и аккаунта.

## Используемые Технологии

*   **QML (Qt Modeling Language)**: Декларативный язык для построения пользовательских интерфейсов.
*   **Qt Framework**: Используется для разработки кроссплатформенных приложений, предоставляя базовые классы и инструментарий.
*   **JavaScript**: Для реализации логики пользовательского интерфейса в QML.
